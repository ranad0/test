# Driving Tone / نغمة القيادة — Luxury Mobility Website

Production-ready static React/Vite/Tailwind website for Driving Tone luxury car rental in Saudi Arabia.

Core idea: **نغمة الوصول / The Tone of Arrival**  
Positioning: **Saudi luxury mobility concierge, not a generic car rental marketplace.**

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- React Router `HashRouter` for GitHub Pages-safe routing
- Static editable data files
- No backend, database, or paid API

## GitHub Pages base path

The current target URL is:

```text
https://ranad0.github.io/test/
```

The default Vite base is set in:

```text
vite.config.ts
```

```ts
base: process.env.VITE_BASE_PATH || '/test/'
```

To deploy under another repository path, change `/test/` or build with:

```bash
VITE_BASE_PATH=/your-repo-name/ npm run build
```

For a custom domain at the root, use:

```bash
VITE_BASE_PATH=/ npm run build
```

## GitHub Pages deployment

This project includes:

```text
.github/workflows/deploy.yml
```

Steps:

1. Push the project to a GitHub repository.
2. Go to `Settings` → `Pages`.
3. Set **Source** to **GitHub Actions**.
4. Push to `main` or run the workflow manually.
5. GitHub Actions builds `dist/` and deploys it to Pages.

The app uses `HashRouter`, so vehicle and service detail pages work on GitHub Pages without server rewrites.

## Fleet imagery

Real fleet photos are stored in:

```text
public/fleet/
```

Every vehicle folder follows this structure:

```text
hero.webp
exterior-1.webp
exterior-2.webp
exterior-3.webp
rear.webp
interior-1.webp
interior-2.webp
interior-3.webp
```

To add or replace a vehicle image:

1. Put the optimized WebP image in the vehicle folder.
2. Keep the same filename if replacing an existing image.
3. Update `src/data/fleet.ts` only if you change folders, names, slugs, or gallery order.
4. Use descriptive Arabic/English alt copy in the `alt` field.

To add a new vehicle:

1. Create `public/fleet/new-vehicle-slug/` with the eight WebP files above.
2. Add a new object in `src/data/fleet.ts`.
3. Include `id`, `slug`, Arabic/English names, year, brand, model, color, category, descriptions, features, best use cases, WhatsApp copy, and SEO fields.

## Design rules

- Arabic-first and RTL-first
- Dark luxury showroom look
- Black, graphite, ivory, champagne gold, bronze accents
- Real vehicle photography dominates the UI
- WhatsApp is the primary conversion path
- Prices are not displayed; pricing is confirmed by date, duration, and location
- No fake reviews, discount-first language, or marketplace clutter
